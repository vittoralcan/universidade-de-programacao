Quem Somos

Bem-vindos à Universidade UPB, a instituição líder em formação excepcional em programação. Objetivos

A Universidade UPB tem como principal objetivo oferecer uma educação de excelência em programação, capacitando os estudantes com habilidades avançadas necessárias para prosperar na era digital. Com uma abordagem integrada de teoria e prática, a instituição visa formar profissionais preparados para enfrentar os desafios tecnológicos contemporâneos. Equipada com laboratórios de ponta e uma equipe dedicada, a UPB está empenhada em preparar os alunos para se destacarem na indústria de tecnologia, promovendo inovação, criatividade e liderança no mundo digital em constante evolução.



A Universidade Conta Com Uma Grande Variedade de Cursos

HTML 5

HTML5 é a quinta e última versão do HTML, sigla de Hypertext Markup Language, uma das primeiras e principais linguagens de marcação da web. O HTML serve, basicamente, para o desenvolvimento de páginas que contenham conteúdos simples, como textos, imagens, áudio, vídeos e etc.

CSS3

CSS3 é a terceira mais nova versão das famosas Cascading Style Sheets, pela qual se define estilos para um projeto web. Com efeitos de transição, imagem, imagem de fundo/background e outros, pode-se criar estilos únicos para seus projetos web, alterando diversos aspectos de design no layout da página.

JavaScript

JavaScript é uma linguagem de programação interpretada estruturada, de script em alto nível com tipagem dinâmica fraca e multiparadigma. Juntamente com HTML e CSS, o JavaScript é uma das três principais tecnologias da World Wide Web.
